======= Personal setup:
workbench.action.terminal.focus				-> cmd + t	(remove When clause)
workbench.action.focusActiveEditorGroup		-> cmd + e
editor.action.indentLines					-> alt + right arrow
editor.action.outdentLines					-> alt + left arrow
editor.action.moveLinesDownAction			-> alt + down arrow
editor.action.moveLinesUpAction				-> alt + up arrow
editor.action.commentLine					-> cmd + /	(also works for uncommenting line, or lines)
editor.action.indentationToTabs				-> ctrl + cmd + t
editor.action.indentationToSpaces			-> ctrl + cmd + s
